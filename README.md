# claude-plugins

Custom Claude Code plugins.

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [che-docs-from-pr](plugins/che-docs-from-pr) | Generate che-docs AsciiDoc articles from GitHub PR code changes |

## Installation

1. Add the marketplace:
   ```
   /plugin marketplace add tolusha/claude-plugins
   ```

2. Install a plugin:
   ```
   /plugin install che-docs-from-pr@claude-plugins
   ```
